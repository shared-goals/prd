### 🟡 Seed Network — подтверждение у Сергея

**Вопрос:** Какая группа станет seed для MVP?

**Контекст (из Текста p2-180):** «Шило в Ж...» — реальное сообщество с нужной механикой: данное слово весомо, совместное преодоление, разный состав (Шило-Девочки, Детское Шило).

### 🟡 Права создателя цели на модерацию коммитов

**Вопрос:** Должен ли создатель public-цели иметь право удалить неуместный коммит участника?

**Варианты:**
- **А.** Нет — создатель не имеет власти над действиями других. Принцип Shared Goals: нет иерархии.
- **Б.** Да, с ограничением — только форматное удаление (спам/флуд), не по содержанию. Решение логируется.
- **В.** Да, с апелляцией — участник может обжаловать через систему модерации.

Предложение: **вариант Б** как минимально достаточный для MVP.

*[на подтверждении у Сергея]*
